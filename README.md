# Bento Plus Classic

## Addon for World of Warcraft Classic with simple and resource-efficient user experience enhancements.

### Features
- **Auto Looting**: Speeds up auto looting and hides the LootFrame.
- **Sound Muting**: Mutes specific in-game sounds (e.g., bow, gun, Mechastrider mount).
- **CVars Adjustments**: Adjusts CVars for better performance and clarity.

### Slash Commands
- `/c`: Closes all temporary chat tabs.
- `/rc`: Performs a ready check.
- `/q`: Leaves the current party or raid.
- `/lua`: Toggles the display of LUA errors.
- `/f KEYWORD`: Filters all active channels for KEYWORD and reposts matching messages.
- `/f KEYWORD1+KEYWORD2`: Filters all active channels for the combination of KEYWORD1 and KEYWORD2 and reposts matching messages.
- `/f`: Clears and stops the filtering.
- `/bc MESSAGE`: Broadcasts the MESSAGE across all joined channels.
- `/bc N1-N2 MESSAGE`: Broadcasts the MESSAGE across specified channels (N1 to N2).
- `/ww MESSAGE`: Sends the MESSAGE to all players in a currently open /who instance.
- `/ww N MESSAGE`: Sends the MESSAGE to the first N players in a currently open /who instance.
- `/ww -CLASS MESSAGE`: Sends the MESSAGE to all players not of the specified CLASS in a currently open /who instance.
